HOW TO USE SPLIT

Passengers traveling using split train tickets travel exactly the same way they woud have had they been traveling on a reguar through train ticket. The only difference is that by splitting their tickets they may have paid a lot less for the same journey. â€‹

You still travel on the same train, at the same times and in the same seat but with two or more cheaper tickets instead of one expensive ticket. The only condition when using split train tickets is that the train you take must stop at all the places you have bought tickets for and not just pass through them.

Using split is easy.On the home page:

1. Input which station code you are travelling from

2. Input which station code you are travelling to.

*Find the station codes here

3. Click Submit and away you go!


This site uses Train fares data under licence from RSP (JAN 2020)

This website shows single fares that are available on the day of travel. In many cases (but not all), return, advance or season tickets will be cheaper. It can be even cheaper to split these as well.
